# AI Chatbot

A Streamlit chatbot powered by Groq and Llama 3.

## Features

* Chat interface
* Multi-turn conversation memory
* Groq API integration
* Environment variable support using .env
* Clear chat button

## Tech Stack

* Python
* Streamlit
* Groq API
* python-dotenv

## Run Locally

```bash
uv sync
streamlit run main.py
```
